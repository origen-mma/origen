# FAR Calibration Distribution Data

Pre-computed spatial probability distributions from RAVEN FAR calibration across the O4 BNS+NSBH population.

## Files

### `far_calibration_fermi_gbm.dat`

Spatial probability distributions for Fermi-GBM (13.2° error radius).

- **Size**: 3.7 MB
- **Format**: Plain text, two columns: `type spatial_prob`
- **Content**:
  - 178 signal trials (true GW-GRB associations)
  - 178,000 background trials (random coincidences)

### `far_calibration_swift_bat.dat`

Spatial probability distributions for Swift-BAT (0.033° = 2 arcmin error radius).

- **Size**: 3.7 MB
- **Format**: Plain text, two columns: `type spatial_prob`
- **Content**:
  - 178 signal trials (true GW-GRB associations)
  - 178,000 background trials (random coincidences)

## Generation

These files are generated by the Rust test in `crates/mm-correlator/src/spatial.rs`:

```bash
cargo test -p mm-correlator test_o4_population_far_calibration -- --ignored --nocapture
```

**Process**:
1. Load 178 BNS+NSBH events from O4 observing scenario
2. For each event:
   - Calculate signal spatial probability (GRB at true position + random offset within error circle)
   - Calculate 1,000 background spatial probabilities (random sky positions)
3. Output distributions to `/tmp/far_calibration_*.dat`

**Runtime**: ~5 minutes (178 events × 2 instruments × 1,001 correlations each)

## Usage

These distribution files are used by plotting scripts to create comparison figures:

```bash
python3 scripts/analysis/plot_instrument_comparison.py
```

They can also be used to:
- Sample realistic spatial probabilities for simulations
- Calibrate FAR thresholds for correlator
- Generate ROC curves for instrument comparison

## Statistics

### Fermi-GBM (13.2° error)
- Signal median: 0.1118
- Background median: 1.3×10⁻⁶
- Discrimination ratio: 86,000×

### Swift-BAT (0.033° error)
- Signal median: 2.7×10⁻⁵
- Background median: 0.0 (69% of trials have zero probability)
- Discrimination ratio: 7,650,000×

## File Format

```
# type spatial_prob
signal 0.88750478
signal 0.88202736
signal 0.85080247
...
background 0.00000123
background 0.00000000
...
```

- **Header**: `# type spatial_prob`
- **Signal lines**: `signal <probability>` (sorted descending)
- **Background lines**: `background <probability>` (sorted descending)
- **Precision**: 8 decimal places
