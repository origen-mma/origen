📚 Usage Instructions

  1. Make sure the API server is running: cargo run --bin mm_api_server --release -- --bind 0.0.0.0:8080
  2. Start the simulation: cargo run --bin stream_o4_simulation --release
  3. Open this HTML file in your browser
  4. Select an event from the dropdown and click "Load Skymap"
  5. Use mouse to pan/zoom the sky view
